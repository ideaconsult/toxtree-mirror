This is ToxTree 1.12 plugin, implementing decision tree for estimating skin irritation and corrosion potential.
Implementation is based on rules publlished in 
The Skin Irritation Corrosion Rules Estimation Tool (SICRET), John D. Walker, Ingrid Gerner, Etje Hulzebos, Kerstin Schlegel, QSAR Comb. Sci. 2005, 24, pp378-384

Run ant to build the plugin. The jar will be found in dist subfolder.

Note (Verhaar scheme plugin should be already build and present in verhaar/dist folder)
