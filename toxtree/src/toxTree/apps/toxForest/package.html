Toxforest 1.18 - multiple decision trees. Toxforest is a full-featured and flexible user-friendly open source application, which is able to define and search complex substructures by hierarchically arranging substructure queries. The software is build upon Toxtree 1.20, with additional features:
  • Able to store, edit and apply multiple decision trees at once.
  • User interface, reflecting Toxforest specifics.
Cheminformatics functionality in {@link toxTree.apps.ToxTreeApp} relies on The Chemistry Development Kit
